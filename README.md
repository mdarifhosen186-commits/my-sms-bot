# 📱 SMS Refer Bot - Telegram Bot

একটি শক্তিশালী Telegram বট যা রেফারেল সিস্টেম ও SMS পাঠানোর সুবিধা প্রদান করে।

## ✨ Features

- 🔗 রেফারেল সিস্টেম (প্রতি রেফারে ৫ পয়েন্ট)
- 📱 SMS পাঠানো (১ পয়েন্ট = ১ SMS)
- 🔐 Force Join (চ্যানেল ও গ্রুপ)
- 👨‍💼 Admin Panel
- 📊 Statistics Dashboard
- 📢 Broadcast SMS to all users
- 📜 SMS History tracking
